# Integrations

Nimbus connects to the tools your team already uses. Set them up from Settings → Integrations, then authorize the connection.

Available integrations:

- **Slack** — get task and project notifications in channels, and create Nimbus tasks from messages.
- **Google Drive** — attach and preview Drive files on tasks.
- **GitHub** and **GitLab** — link commits and pull requests to tasks and auto-update status.
- **Figma** — embed live design previews on tasks.
- **Zoom** — schedule and join project calls from Nimbus.
- **Jira** — two-way sync of issues for teams migrating or running both tools.
- **Zapier** — connect Nimbus to 6,000+ other apps with no code.
- **REST API & webhooks** — build custom integrations and react to events in real time.

**Plan availability:** Slack and Google Drive are available on every plan, including Starter (Free). GitHub, GitLab, Figma, Zoom, Jira, Zapier, and the REST API & webhooks require Pro or higher. SSO/SAML is a Business feature.

Each integration can be connected once per workspace and managed by workspace admins.
