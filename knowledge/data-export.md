# Exporting Your Data

You can export your workspace data at any time, on any plan, including Starter (Free). Workspace admins can run an export from Settings → Data & Privacy → Export data.

Export formats:

- **CSV** — tasks and projects as spreadsheets, ideal for analysis in Excel or Google Sheets.
- **JSON** — a complete structured export of the workspace (projects, tasks, comments, and metadata), ideal for migrating or backing up.

File attachments are included as a separate ZIP archive.

For small workspaces the export downloads immediately. For larger workspaces we generate it in the background and email a secure download link, usually within a few hours. That link is valid for 7 days.

Exports reflect the data at the moment you start them. If you need a personal data report for a GDPR or CCPA request, the same export covers it; contact support if you need anything additional or a signed data-processing agreement.
